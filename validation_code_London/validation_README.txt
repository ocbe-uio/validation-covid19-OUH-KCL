Validation of the models in Xie et al., 2020; Zhang et al., 2020; Allenbach et al., 2020
----------------------------------------------------------------------------------------

All variables needed for validation of Zhang, Xie and Allenbach
should be included in a single excel file named "data.xlsx"

The following variables should be stored in the excel file
"variable name": description (unit of measure)

"death" : patient death (1=death)
"poor_outcome": defined in Zhang et al., pag. 6 (1=poor outcome)
"deathICU" : death or ICU by day 14 (1=death/ICU)
"age": age (years)
"gender": gender (1=male)
"lymp": lymphocyte count (10^9/L or 10^3/mm^3)
"ld": lactate dehydrogenase, LDH (U/L)
"spo2": peripheral capillary oxygen saturation, SpO2 (%)
"crp": C-reactive protein, CRP (mg/L)
"neut": neutrophil count (10^9/L)
"plt": platelet count (10^9/L)
"crea": creatinine (mu-mol/L)
"WHOscale": WHOscale - see Allenbach et al., 2020 (points)
